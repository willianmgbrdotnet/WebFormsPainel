# WebFormsPainel

## Panel (Painel)

O controle do painel é usado como um contêiner para outros controles. <br />
Um controle de painel é muito útil quando você deseja agrupar controles e, em seguida, mostrar ou ocultar todos os controles do grupo. <br />
O controle do painel também é muito útil ao adicionar controles ao formulário da web de forma dinâmica.
